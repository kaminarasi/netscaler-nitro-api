###update



URL: http://&lt;NSIP&gt;/nitro/v1/config/
HTTP Method: PUT
Request Payload: ```{"params": {      "warning":<String_value>,      "onerror":<String_value>"},sessionid":"##sessionid","snmpalarm":{      "trapname":<String_value>,      "thresholdvalue":<Double_value>,                  "normalvalue":<Double_value>,      "time":<Double_value>,      "state":<String_value>,      "severity":<String_value>,      "logging":<String_value>,}}```
Response Payload: 
{ "errorcode": 0, "message": "Done", "severity": <String_value> }
